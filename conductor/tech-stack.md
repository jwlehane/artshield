# Tech Stack: ArtShield

## 1. Frontend
-   **Framework:** React
-   **Build Tool:** Vite
-   **Language:** TypeScript
-   **Styling:** Tailwind CSS
-   **UI Library:** Radix UI

## 2. Backend
-   **Framework:** FastAPI
-   **Language:** Python
-   **ML/AI Libraries:** PyTorch
-   **Image Processing:** Pillow
-   **Packaging:** PyInstaller

## 3. DevOps & Tools
-   **Version Control:** Git
-   **CI/CD:** GitHub Actions
